This project must be used with the Smart card WCF sercice using the Net TCP binding.

It is a Windows Store App that behaves similarly to the ExchangeAPDU application of the Smartcard Framework project. This application is a demonstration of how to be able to access a smart card resource from a Windows Store app which is not possible with the standard WinRT API.

The content of this code is presented in the following CodeProject article:

http://www.codeproject.com/Articles/540839/Using-Smart-cards-with-a-Windows-Store-App
